# Terminal Portfolio

A terminal-style portfolio site with a “hacker” feel.

## Run (XAMPP)

- Put this folder inside `htdocs` (you already did).
- Start Apache in XAMPP.
- Open `http://localhost/portfolio/`

## Usage

- First input: your visitor’s name → prompt becomes `name@portfolio:~$`
- Commands:
  - `help`
  - `about`
  - `projects` (then `1`, `2`, or `3`)
  - `contact`
  - `socials`
  - `clear` / `cls`

## Customize

Edit `app.js`:

- Update `content.about()` with your real “about me”
- Update `content.projects()` and `content.projectDetails()`
- Update `content.contact()` and `content.socials()`
